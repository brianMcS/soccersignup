Source of truth

Game details come from the organiser.

The organiser sets the date, kick-off time, location, cost, and player limit for each game. Those details can change, so the current game page is the place to check before signing up.

  • Availability may be open, full, waitlisted, or closed.
  • Costs and payment links are shown only when set for the game.
  • Team size depends on the game the organiser creates.
